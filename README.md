---

unityちゃんが働いてマイホームを買います。

　　　　操作：WSキーで移動。ADキーで回転。
    
　　　　　　　クリックで決定。
       
　　　ルール：仕事に行って、薪切りをします。
   
　　　　　　　稼いだお金で家を買います。
       
ギリギリ要素：薪をギリギリしますｗ

　　　　注意：ロードが非常に長いです。
    
　　　　　　　家の更新がおかしいです。

ここだけの話。「money」を同時に押すと。。。。。
